# custom-bike-computer

I am making this as a low power bike computer. It will use an e-ink display for power efficiency. I'm also making a cadence sensor to go with it, which will connect with bluetooth. If I have time, I will also have it read heart rate data from any HRM, including watches that can broadcast heart rate. The bike computer will have a custom PCB so vibrations during rides are less likely to disrupt wiring.
